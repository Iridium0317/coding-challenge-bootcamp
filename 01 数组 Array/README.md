# 数组 Array
数组下标都是从0开始的。 数组内存空间的地址是连续的。
*   查 ： $O(1)$。
    *   这是数组最强的地方。只要知道下标 `i`，通过 `内存起始地址 + i * 元素大小` 瞬间就能定位。
    *   面试陷阱： “查找”是指 `nums[i]`。如果你要“搜索”某个值（`if target in nums`），那是 $O(N)$。

*   增/删 (Insert/Delete)：$O(N)$ 。
    *   如果你想在数组中间删除一个元素，后面的所有元素都要向前移动一位，把坑填上。
    *   如果你想在中间插入一个元素，后面的所有元素都要向后移动一位，腾出位置。
    *   注意： `nums.pop(0)` 或者 `nums.remove(val)` 都是 $O(N)$ 的操作。
