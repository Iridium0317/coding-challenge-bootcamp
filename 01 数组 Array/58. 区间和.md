# 58. Interval Sum (区间和)
[KamaCoder Link](https://kamacoder.com/problempage.php?pid=1070)

**Problem**  
Given an integer array (represented by `n` followed by `n` integers), and a series of query intervals `a` and `b`, calculate the sum of the elements in the range `[a, b]` for each query.

**Input Description**
*   The first line contains an integer `n`, the length of the array.
*   The next `n` lines each contain an integer, representing the elements of the array.
*   The following lines until the end of the file each contain two integers `a` and `b`, representing the query interval `[a, b]`.

**Output Description**
*   For each query interval `a` and `b`, output the sum of the elements in that range.

**Example**

```text
Input:
5
1
2
3
4
5
0 1
1 3

Output:
3
9

Explanation:
The array is [1, 2, 3, 4, 5].
For query 0 1: sum is 1 + 2 = 3.
For query 1 3: sum is 2 + 3 + 4 = 9.
```

**Constraints:**
*   `0 < n <= 100,000`
*   `-1000 <= value <= 1000`
*   `0 <= a <= b < n`

## My Understanding

### Note
```text
n = int(next(iterator))
Python 处理流式数据（文件输入、网络数据或大规模列表）的经典写法
从数据流里拿出下一个东西。
把拿出来的东西（通常是字符串）变成整数。
赋值给 n。
```

### Solution
```python
import sys

def main():
    input_data = sys.stdin.read().split()
    
    if not input_data:
        return

    iterator = iter(input_data)
    
    try:
        # 2. 解析第一行：数组长度 n
        n = int(next(iterator))
        nums = []

        for _ in range(n):
            nums.append(int(next(iterator)))
            
        presum = [0] * n
        current_sum = 0
        
        for i in range(n):
            current_sum += nums[i]
            presum[i] = current_sum
        
        results = []
        while True:
            try:
                a = int(next(iterator))
                b = int(next(iterator))
                
                if a == 0:
                    sum_val = presum[b]
                else:
                    sum_val = presum[b] - presum[a - 1]
                
                results.append(str(sum_val))
            except StopIteration:
                break

        print('\n'.join(results))
        
    except StopIteration:
        pass

if __name__ == "__main__":
    main()
```

# 58. Length of Last Word 

[LeetCode Link](https://leetcode.cn/problems/length-of-last-word/)

**Problem**  
Given a string `s` consisting of words and spaces, return *the length of the **last** word in the string*.

A **word** is a maximal substring consisting of non-space characters only.

**Example**

```text
Input: s = "Hello World"
Output: 5
Explanation: The last word is "World" with length 5.

Input: s = "   fly me   to   the moon  "
Output: 4
Explanation: The last word is "moon" with length 4.

Input: s = "luffy is still joyboy"
Output: 6
Explanation: The last word is "joyboy" with length 6.
```

**Constraints:**
*   1 <= s.length <= 10^4
*   `s` consists of only English letters and spaces `' '`.
*   There will be at least one word in `s`.

## My Understanding

### Note
```text
从后往前遍历
```

### Solution
```python
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        length = 0
        for i in range(len(s) - 1, -1, -1):
            if s[i] == ' ':
                if length > 0:
                    return length
            else:
                length += 1
            
        return length
```
**Complexity**

*   Time Complexity: $O(N)$
*   Space Complexity: $O(1)$
